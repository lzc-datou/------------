1、按 Alt + Shift + ‘=’ 扩展选区，按 Alt + Shift + ‘-’ 缩小选区
2、按 Ctrl + 双击窗口标题：将窗口停靠到它之前停靠的位置
3、多行编辑代码的技巧，这对将相同的更改应用于多行非常有用。下面有 4 个多行编辑代码的技巧：（这个技巧简直无敌）
	1、按住 Alt 键，使用鼠标拖选
	2、使用多个光标同时编辑。按住 Ctrl + Alt 键，点击或选择要插入或编辑的位置
	3、匹配相同单词并编辑。先选择要编辑的单词或字符，再按一下 Alt + Shift + ；，会自动匹配并选中所有相同的单词，然后开始编辑，你也可以使用 Alt + Shift + . 只匹配选择下一个相同的单词。
4、调试：打断点，开始调试，按F11执行下一句指令，在 调试->窗口 中调出监视窗口和局部变量窗口并放到合适的位置然后在    
  窗口->保存窗口布局   中保存当前布局即可使得下次调试时自动是设置好的布局
5、运行按钮： 一个调试运行按钮，一个不调试直接运行的按钮
6、注释：直接点击  注释  和  取消注释  按钮即可
7、右击鼠标可以选择转到声明，转到定义，查找所有引用等功能，十分方便的跳转位置，并且点击左上角的向左的箭头可以返回上一个位置，点击向右箭头返回下一个位置
8、点击在文件中查找按钮可以十分方便的在文件中查找特定信息。
9、ctrl + k,ctrl + d 整理代码
10、	i. Ctrl-M-O 折叠所有方法
	ii. Ctrl-M-P 展开所有方法并停止大纲显示（不可以再折叠了）
	iii. Ctrl-M-M 折叠或展开当前方法
	iv. Ctrl-M-L展开所有方法
11、	Ctrl + Enter = 在当前行插入空行
	Ctrl + Shift + Enter =在当前行下方插入空行
	Fn + end键=光标跳到行尾
	Fn + home键 = 光标跳到行首
12、tab键自动补全代码
13、Ctrl +Up/Down = 滚动窗口但不移动光标
14、Shift+Alt+Enter：最大化代码编写区域（代码全屏模式），即去掉所有其它辅助窗口只留下代码编写窗口，再按一次返回到原来界面
